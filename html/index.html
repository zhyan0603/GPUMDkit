    GPUMDkit 是什么？

    • 为 GPUMD 与 NEP 提供统一命令入口。
    • 支持 格式转换、结构采样、工作流自动化、计算器、分析器、绘图脚本。
    • 具备交互式菜单与命令行双模式，适配初学者与高通量用户。
    • 支持自定义命令与 Bash 补全，便于个人化扩展。

    安装与快速开始

    git clone https://github.com/zhyan0603/GPUMDkit.git
cd GPUMDkit
source ./install.sh
source ~/.bashrc
    • 推荐在独立 Python 环境安装增强依赖（如 neptrain / ase / pymatgen / dpdata）。
    • 启动：gpumdkit.sh（交互模式）或 gpumdkit.sh -h（命令行模式）。

    两种使用模式

    交互模式

    适合新手与复杂流程，按菜单编号逐步执行。

    gpumdkit.sh

    命令行模式

    适合批处理与自动化脚本，直接执行子命令。

    gpumdkit.sh -plt train

    # 常见命令
gpumdkit.sh -out2xyz .
gpumdkit.sh -calc ionic-cond Li 1
gpumdkit.sh -range train.xyz force

    教程路径建议（新用户）

    1. 安装并运行 gpumdkit.sh -h 熟悉命令总览。
    2. 先做格式转换：OUTCAR/POSCAR/CIF/LAMMPS ↔ extxyz。
    3. 进入采样与过滤：构建更高质量训练数据。
    4. 使用 calculators + analyzer 做性质计算与质量检查。
    5. 使用 -plt 系列完成训练和模拟结果可视化。
    6. 最后尝试 workflow 与 active learning 自动化。
