GPUMDkit

A Toolkit for GPUMD and NEP

GPUMDkit is a user-friendly toolkit for the GPUMD (Graphics Processing Units Molecular Dynamics) and NEP (Neuroevolution Potential) program. It provides a set of tools to streamline the use of common scripts in GPUMD and NEP, simplifying workflows and enhancing efficiency.

Get Started

Key Features

🔄
Format Conversion

Seamlessly convert between POSCAR, XYZ, CIF, and other atomic structure formats.

📦
Sample Structures

Generate and sample structures for NEP training.

📊
Data Analysis

Built-in tools for analyzing trajectories, calculating properties, and visualizing simulation outcomes.
