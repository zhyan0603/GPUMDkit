GPUMDkit

GPUMD 与 NEP 工具包

GPUMDkit 是一款面向 GPUMD（Graphics Processing Units Molecular Dynamics）和 NEP（Neuroevolution Potential）程序的用户友好工具包。它提供用户友好的命令行界面，涵盖脚本调用、格式转换、结构采样、NEP 构建流程以及各类分析与可视化功能，并提供交互模式和命令行模式两种使用方式，极大简化了工作流程，旨在提升用户工作效率。
