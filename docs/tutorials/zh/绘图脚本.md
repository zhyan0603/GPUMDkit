<div align="center">
  <h1>📊 绘图脚本</h1>
  <p style="text-align: justify;">绘图模块用于可视化 GPUMD、NEP 和 GPUMDkit 计算器产生的数据。</p>
</div>

**脚本位置：** `Scripts/plt_scripts/`

## 快速入口

```bash
gpumdkit.sh -plt
gpumdkit.sh -plt <type>
gpumdkit.sh -plt <type> save
gpumdkit.sh -plt <type> -h
```

## NEP 训练与预测

### 训练结果

读取 `loss.out`、`energy_train.out`、`force_train.out`、`virial_train.out`，绘制 loss 和能量/力/virial 的 parity plot。

```bash
gpumdkit.sh -plt train
```

## 快速表

| 命令 | 输入文件 | 用途 |
|------|----------|------|
| `train` | `loss.out`, `*_train.out` | NEP 训练结果 |
| `prediction` | `*_test.out` | NEP 预测结果 |
| `thermo` | `thermo.out` | 热力学量 |
| `msd` | `msd.out` | 均方位移 |
| `sdc` | `msd.out` | 自扩散系数 |
| `rdf` | `rdf.out` | 径向分布函数 |
| `pdos` | `dos.out`, `mvac.out` | VAC 和 PDOS |
| `emd`/`nemd`/`hnemd` | 热输运输出 | 热导率分析 |
| `des` | `descriptors.npy` | 描述符降维 |
| `dimer` | `nep.txt` | 二聚体能量/力曲线 |
| `doas` | `doas.out` | 原子态密度 |
