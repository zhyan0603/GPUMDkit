## 1. 准备 Python 环境

可以使用干净的 conda 环境：

```bash
conda create -n gpumdkit python=3.12
conda activate gpumdkit
```

部分可选功能需要额外依赖：

```bash
pip install neptrain dpdata calorine
```

其他 Python 依赖由对应脚本在运行时导入。如果缺少某个包，使用对应功能时 Python 会给出提示。

## 2. 安装 GPUMDkit

克隆仓库：

```bash
git clone https://github.com/zhyan0603/GPUMDkit.git
cd GPUMDkit
```

运行安装脚本：

```bash
source ./install.sh
```

安装脚本会写入 `GPUMDkit_path` 和 `PATH` 设置，并在当前 shell 中加载。默认写入 `~/.bashrc`；如果当前 shell 是 zsh，则写入 `~/.zshrc`。如果检测到已有 GPUMDkit 路径，安装脚本会打印旧路径并询问是否替换；修改 rc 文件前会先创建备份。

### 命令行模式

直接命令使用固定的位置参数：

```bash
gpumdkit.sh -pos2exyz POSCAR model.xyz
gpumdkit.sh -plt train
gpumdkit.sh -plt thermo
```

例如：

```bash
gpumdkit.sh -pos2exyz POSCAR model.xyz
```

表示读取 `POSCAR`，写出 `model.xyz`。

## 4. 实际例子

### POSCAR 转 extxyz

```bash
gpumdkit.sh -pos2exyz POSCAR model.xyz
```

### 添加 GPUMD group 标签

```bash
gpumdkit.sh -addgroup POSCAR Li Y Cl
```

group 标签用于一些需要原子分组的 GPUMD 相关流程和分析，例如特定元素的 MSD 或扩散性质计算。
