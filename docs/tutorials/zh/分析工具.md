**脚本位置：** `Scripts/analyzer/`

## 功能概览

| 任务 | 命令 | 用途 |
|------|------|------|
| 组分分析 | `gpumdkit.sh -analyze_comp train.xyz` | 按化学组分统计结构 |
| 元素列表 | `gpumdkit.sh -chem_species train.xyz` | 查看文件中有哪些元素 |
| 性质范围 | `gpumdkit.sh -range train.xyz force` | 检查能量/力/virial 范围 |
| 最小距离 | `gpumdkit.sh -min_dist dump.xyz` | 快速检查，不考虑 PBC |
| PBC 最小距离 | `gpumdkit.sh -min_dist_pbc dump.xyz` | 考虑 PBC 的距离检查 |
| 电荷平衡 | `gpumdkit.sh -cbc train.xyz` | 检查常见氧化态是否平衡 |
| 距离过滤 | `gpumdkit.sh -filter_dist_pbc dump.xyz 1.0` | 删除短距离结构 |
| 盒子过滤 | `gpumdkit.sh -filter_box dump.xyz 13` | 删除盒子边长过大的结构 |
| 数值过滤 | `gpumdkit.sh -filter_value train.xyz force 20` | 按性质阈值过滤 |
| 元素对距离 | `gpumdkit.sh -filter_range dump.xyz Li Li 1.8 2.0` | 按元素对距离范围提取 |

## 组分分析

```bash
gpumdkit.sh -analyze_comp train.xyz
```

示例输出：

```text
Index    Compositions           N atoms      Count
---------------------------------------------------
1        Li56O96Zr16La24        192          51
---------------------------------------------------
Enter index to export (e.g., '1,2', '2-3', 'all'), or press Enter to skip:
```

如果一个 `train.xyz` 中混合了多个体系或不同超胞，这个功能很有用。

## 能量/力/virial 范围

```bash
gpumdkit.sh -range train.xyz force
gpumdkit.sh -range train.xyz energy
gpumdkit.sh -range train.xyz virial
gpumdkit.sh -range train.xyz force hist
```

使用 `hist` 可以画出所选性质的直方图。

## 最小距离检查

快速版本：

```bash
gpumdkit.sh -min_dist dump.xyz
```

考虑 PBC 的版本：

```bash
gpumdkit.sh -min_dist_pbc dump.xyz
```

PBC 版本会考虑周期性边界条件。

## 结构过滤

按最小距离过滤：

```bash
gpumdkit.sh -filter_dist_pbc dump.xyz 1.0
```

按盒子大小过滤：

```bash
gpumdkit.sh -filter_box dump.xyz 13
```

按性质值过滤：

```bash
gpumdkit.sh -filter_value train.xyz force 20
```

按元素对距离范围提取：

```bash
gpumdkit.sh -filter_range dump.xyz Li Li 1.8 2.0
```

## 电荷平衡检查

```bash
gpumdkit.sh -cbc train.xyz
```

可能输出：

- `balanced.xyz`
- `unbalanced.xyz`
- `indices.txt`

这个功能适合常见氧化态有意义的离子材料。

## 示例命令

性质和距离检查：

```bash
gpumdkit.sh -analyze_comp train.xyz
gpumdkit.sh -range train.xyz force hist
gpumdkit.sh -range train.xyz energy hist
gpumdkit.sh -min_dist_pbc train.xyz
```

过滤示例：

```bash
gpumdkit.sh -filter_dist_pbc dump.xyz 1.0
gpumdkit.sh -filter_box filtered_dist.xyz 13
```
